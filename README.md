# qID
qID - Quick Image develop | A simple image editing program by python for webmaster.

## Screenshot
![qID](https://maxmacstn.files.wordpress.com/2017/05/1.png)

## Features
* Basic Adjustments
  * Brightness
  * Contrast
  * Saturation
* Instant Filter
* Set width of exported photos
* Sharpen
* Watermark
* Create and load preset
